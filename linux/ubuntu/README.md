* How to set root password? 
  ```
  sudo passwd root
  ```
* How to switch to root user from general user?
  * 1st way: su root
  * 2nd way: su
  * 3rd way: sudo su
* How to switch to general user from root user?
  * su yang
* How to view package list?
  * 1st way: dpkg -l | grep '^ii'
  * 2nd way
    * sudo apt-get install aptitude
    * sudo aptitude
